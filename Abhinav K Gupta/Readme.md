# **Introduction**
This is the Web Application to authenticate the website using Node.js, Express, Mongoose, EJS and express-sessions.

# **Prerequisites**
You need to have installed Node.js, NPM and MongoDB in your System.
# **Installing**
npm install
# To Run
nodemon server.js
# **Code is Running on**

http://localhost:3000/
